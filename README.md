# Qiming Guo - Academic Homepage

Personal academic homepage of Qiming Guo, a cybersecurity researcher specializing in robotic kernel security at Beihang University (BUAA), NKLSTISS Lab.

## Live Site
[https://grubwithu.github.io](https://grubwithu.github.io)

## Features

- **Clean, Minimal Design**: Professional academic homepage with focus on content
- **Responsive Layout**: Works on all devices from mobile to desktop
- **Fast Loading**: Optimized for performance with minimal dependencies
- **Easy to Update**: Simple Markdown-based content management
- **SEO Optimized**: Proper meta tags and semantic HTML

## Pages

- **Home**: Personal introduction, research interests, education overview
- **Publications**: Research papers and academic publications
- **Projects**: Research projects and open source contributions
- **Awards**: Academic awards, honors, and recognitions
- **Education**: Detailed academic background and skills

## Technology Stack

- **Static Site Generator**: Jekyll (GitHub Pages compatible)
- **Styling**: Custom CSS with CSS variables for theming
- **Icons**: Font Awesome for vector icons
- **Fonts**: Google Fonts (Inter for body, Source Code Pro for code)
- **Hosting**: GitHub Pages

## Local Development

```bash
# Clone the repository
git clone https://github.com/grubwithu/grubwithu.github.io.git

# Install Jekyll (if not already installed)
gem install bundler jekyll

# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve

# Open in browser
open http://localhost:4000
```

## Customization

1. Update personal information in `_config.yml`
2. Modify content in Markdown files
3. Update styling in `assets/css/style.css`
4. Replace `avatar.jpg` with your own profile picture
